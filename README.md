A full-stack MERN Notes application featuring production-grade authentication and security.

Key features:

•   Email verification with OTP

•   Forgot password & reset flow

•   JWT access & refresh token authentication

•`  Secure password hashing with bcrypt

•   Input validation using Yup for forms & requests

•   Image upload for notes using Multer

•   Protected routes and session handling

•   CRUD Notes & Todo management

Built with MongoDB, Express.js, React, and Node.js following secure authentication best practices.
